# 集合論基礎

集合論是現代數學的基石，它提供了一種描述、分類和操作數學對象的通用語言。從數系到函數，從幾何到拓樸，幾乎所有的數學概念都可以在集合論的框架下被形式化。本章將介紹集合論的核心概念、表示方法、基本關係以及常見錯誤，為您後續學習其他數學領域奠定堅實基礎。

-----

### 1. 核心概念與定義

#### 什麼是集合？

*   **核心觀念：** 集合 (Set) 是一個明確定義的、由不同對象（稱為「元素」或「成員」）組成的整體。集合中的元素沒有特定的順序，且每個元素都是唯一的。

*   **表示法：**
    *   **列舉法 (Roster Method / Listing Method)：** 將集合的所有元素一一列出，用逗號分隔，並用大括號 `{}` 括起來。
        *   **例子：**
            *   所有小於 5 的正整數所組成的集合 $A = \{1, 2, 3, 4\}$。
            *   英文字母中的母音字母集合 $V = \{a, e, i, o, u\}$。
            *   包含數字 1 和集合 $\{2, 3\}$ 作為元素的集合 $S = \{1, \{2, 3\}\}$。
    *   **描述法 (Set-Builder Notation)：** 透過描述元素的共同性質來定義集合。通常形式為 $\{x \mid \text{P}(x)\}$，讀作「所有滿足性質 P(x) 的 x 所組成的集合」。垂直線 `|` 有時也用冒號 `:` 代替。
        *   **例子：**
            *   所有偶數的集合 $E = \{x \mid x \text{ 是偶數}\}$ 或 $E = \{x \mid x = 2k, k \in \mathbb{Z}\}$。
            *   所有大於 0 的實數集合 $R^+ = \{x \in \mathbb{R} \mid x > 0\}$。

*   **與相鄰概念的關聯：** 集合是數學中最基本的概念之一，它為所有其他數學結構提供了基礎。你可以將其視為數學語言中的「名詞」。

#### 元素 (Element)

*   **核心觀念：** 構成集合的對象稱為元素。

*   **符號：**
    *   如果 $x$ 是集合 $A$ 的一個元素，我們寫作 $x \in A$ (讀作「$x$ 屬於 $A$」或「$x$ 是 $A$ 的元素」)。
    *   如果 $x$ 不是集合 $A$ 的元素，我們寫作 $x \notin A$。

*   **例子：**
    *   對於集合 $A = \{1, 2, 3\}$，有 $1 \in A$ 和 $2 \in A$，但 $4 \notin A$。
    *   對於集合 $V = \{a, e, i, o, u\}$，有 $e \in V$，但 $b \notin V$。

*   **與相鄰概念的關聯：** 元素是構成集合的「原子」。判斷一個對象是否為某集合的元素，是理解集合的基礎。

#### 空集合 (Empty Set)

*   **核心觀念：** 空集合是一個不包含任何元素的集合。它是唯一的。

*   **符號：** $\emptyset$ 或 `{}`。

*   **例子：**
    *   所有大於 5 且小於 1 的實數所組成的集合是空集合。
    *   一個正常人類擁有三隻腳的集合是空集合。

*   **性質：** 空集合是任何集合的子集合。

*   **與相鄰概念的關聯：** 空集合是一個特殊的集合，它的存在保證了集合論的完整性和一致性。它在集合運算中扮演類似於數字 0 的角色。

#### 全集合 (Universal Set)

*   **核心觀念：** 在特定討論範圍內，包含所有考慮中元素的集合稱為全集合。全集合並非絕對的，它取決於上下文。

*   **符號：** 通常用 $U$ 表示。

*   **例子：**
    *   如果我們在討論整數的性質，那麼全集合可能是整數集合 $\mathbb{Z}$。
    *   如果我們在討論學生考試成績，那麼全集合可能是所有學生的集合。

*   **與相鄰概念的關聯：** 全集合定義了我們研究的「宇宙」，它對於理解補集等概念至關重要。

#### 子集合 (Subset) 與 真子集合 (Proper Subset)

*   **核心觀念：**
    *   **子集合：** 如果集合 $A$ 的所有元素都是集合 $B$ 的元素，則稱 $A$ 是 $B$ 的子集合。
    *   **真子集合：** 如果 $A$ 是 $B$ 的子集合，且 $A$ 不等於 $B$ (即 $B$ 至少包含一個不在 $A$ 中的元素)，則稱 $A$ 是 $B$ 的真子集合。

*   **符號：**
    *   **子集合：** $A \subseteq B$ (讀作「$A$ 包含於 $B$」或「$A$ 是 $B$ 的子集合」)。
    *   **真子集合：** $A \subset B$ (讀作「$A$ 真包含於 $B$」或「$A$ 是 $B$ 的真子集合」)。

*   **例子：**
    *   設 $A = \{1, 2\}$，$B = \{1, 2, 3\}$，$C = \{1, 2\}$。
        *   $A \subseteq B$ 且 $A \subset B$ (因為 $3 \in B$ 但 $3 \notin A$)。
        *   $A \subseteq C$ (因為 $A$ 和 $C$ 的元素完全相同)。
        *   $A \not\subset C$ (因為 $A$ 不等於 $C$)。

*   **性質：**
    *   空集合是任何集合的子集合：$\emptyset \subseteq A$ 對於任意集合 $A$ 成立。
    *   任何集合都是其本身的子集合：$A \subseteq A$ 對於任意集合 $A$ 成立。

*   **與相鄰概念的關聯：** 子集合關係是集合之間最基本的比較方式之一，它建立了集合之間的層次結構。理解子集合對於後續學習集合運算和證明題非常重要。

#### 集合的相等 (Equality of Sets)

*   **核心觀念：** 兩個集合 $A$ 和 $B$ 如果它們包含完全相同的元素，則稱它們相等。元素的順序或重複出現不影響集合的相等。

*   **判斷：** $A = B$ 若且唯若 $A \subseteq B$ 且 $B \subseteq A$。

*   **例子：**
    *   $\{1, 2, 3\} = \{3, 1, 2\}$。
    *   $\{a, b, b, c\} = \{a, b, c\}$。
    *   $\{x \in \mathbb{N} \mid x < 3\} = \{1, 2\}$。

*   **與相鄰概念的關聯：** 集合的相等性定義了兩個集合何時被視為同一個實體，這對於在不同表示法之間轉換集合以及證明集合關係時非常關鍵。

#### 基數 (Cardinality)

*   **核心觀念：** 集合的基數是指集合中元素的數量。

*   **符號：** 通常用 $|A|$ 或 $n(A)$ 表示集合 $A$ 的基數。

*   **例子：**
    *   如果 $A = \{1, 2, 3, 4\}$，那麼 $|A| = 4$。
    *   如果 $V = \{a, e, i, o, u\}$，那麼 $|V| = 5$。
    *   如果 $E = \emptyset$ (空集合)，那麼 $|E| = 0$。
    *   自然數集合 $\mathbb{N}$ 是一個無限集，其基數為 $\aleph_0$ (讀作 "aleph-naught")。

*   **與相鄰概念的關聯：** 基數將集合的概念與「數量」連結起來。對於有限集，基數是簡單的計數；對於無限集，基數概念引導出對不同「大小」無限的探討。

### 2. 典型例子與轉換/推導

#### 集合的表示法轉換

*   **例子 1：列舉法轉描述法**
    *   給定 $A = \{2, 4, 6, 8\}$。
    *   **轉換：** 可以描述為「小於 10 的正偶數集合」。
    *   **描述法：** $A = \{x \in \mathbb{N} \mid x \text{ 是偶數且 } x < 10 \}$。

*   **例子 2：描述法轉列舉法**
    *   給定 $B = \{x \in \mathbb{Z} \mid -2 < x \le 3 \}$。
    *   **推導：** 尋找介於 -2 和 3 之間（包含 3）的整數。這些整數是 -1, 0, 1, 2, 3。
    *   **列舉法：** $B = \{-1, 0, 1, 2, 3\}$。

#### 子集合的列舉與冪集合 (Power Set)

*   **核心觀念：**
    *   一個集合的所有子集合的集合，稱為該集合的冪集合。
    *   如果一個集合有 $n$ 個元素，那麼它將有 $2^n$ 個子集合。

*   **符號：** 集合 $A$ 的冪集合表示為 $P(A)$ 或 $\mathcal{P}(A)$。

*   **例子：**
    *   設 $A = \{a, b\}$。
    *   **所有子集合：**
        *   不包含任何元素：$\emptyset$
        *   包含一個元素：$\{a\}$, $\{b\}$
        *   包含兩個元素：$\{a, b\}$
    *   **冪集合：** $P(A) = \{\emptyset, \{a\}, \{b\}, \{a, b\}\}$。
    *   **基數：** $|A|=2$，所以 $|P(A)| = 2^2 = 4$。這與我們列出的子集合數量一致。

*   **例子：**
    *   設 $S = \{1, \{2\}\}$。
    *   **注意：** 集合 $S$ 有兩個元素，分別是 $1$ 和 $\{2\}$。
    *   **所有子集合：**
        *   $\emptyset$
        *   $\{1\}$
        *   $\{\{2\}\}$ (這個子集合的元素是 $S$ 的第二個元素 $\{2\}$)
        *   $\{1, \{2\}\}$ (這就是 $S$ 本身)
    *   **冪集合：** $P(S) = \{\emptyset, \{1\}, \{\{2\}\}, \{1, \{2\}\}\}$。
    *   **基數：** $|S|=2$，所以 $|P(S)| = 2^2 = 4$。

### 3. 與相鄰概念的關聯

#### 邏輯 (Logic)

*   集合論與邏輯學有著深刻的聯繫。集合的概念可以用邏輯命題來定義，反之亦然。
*   **元素歸屬：** 「$x \in A$」可以看作一個命題，判斷其真假。
*   **子集關係：** 「$A \subseteq B$」等價於邏輯命語「對於所有的 $x$，如果 $x \in A$，那麼 $x \in B$」。這可以用量詞和蘊含符號表示為 $\forall x (x \in A \implies x \in B)$。
*   **集合運算：** 集合的聯集、交集等運算與邏輯的析取（OR）、合取（AND）等運算有直接的對應關係。例如，$x \in A \cup B$ 等價於 $x \in A \text{ 或 } x \in B$。

#### 數系 (Number Systems)

*   我們在數學中最常使用的數系都是集合的典型例子：
    *   **自然數集合：** $\mathbb{N} = \{1, 2, 3, \ldots\}$ (或 $\{0, 1, 2, \ldots\}$ 依定義而異)
    *   **整數集合：** $\mathbb{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots\}$
    *   **有理數集合：** $\mathbb{Q} = \{\frac{a}{b} \mid a \in \mathbb{Z}, b \in \mathbb{Z}, b \neq 0 \}$
    *   **實數集合：** $\mathbb{R}$ (包含所有有理數和無理數)
    *   **複數集合：** $\mathbb{C}$
*   這些數系之間存在著明確的子集關係：$\mathbb{N} \subseteq \mathbb{Z} \subseteq \mathbb{Q} \subseteq \mathbb{R} \subseteq \mathbb{C}$。
*   學習集合論有助於理解這些數系的結構和它們之間的關係。

#### 函數 (Functions)

*   在更嚴格的定義下，函數可以被視為一種特殊的關係集合。
*   如果 $f$ 是一個從集合 $A$ 到集合 $B$ 的函數，我們可以將 $f$ 定義為一個有序對的集合 $f \subseteq A \times B$ (其中 $A \times B$ 是笛卡爾積，即所有可能的 $(a, b)$ 對組成的集合)，且滿足對於每個 $a \in A$，恰好存在一個 $b \in B$ 使得 $(a, b) \in f$。
*   集合論為函數的定義域 (Domain)、對應域 (Codomain) 和值域 (Range) 提供了清晰的框架。

### 4. 進階內容

#### 羅素悖論 (Russell's Paradox)

*   **核心觀念：** 羅素悖論是一個著名的集合論悖論，它揭示了樸素集合論 (Naive Set Theory) 中存在的問題，並促使了公理化集合論 (Axiomatic Set Theory, 如 ZFC 公理系統) 的發展。
*   **悖論內容：** 考慮一個集合 $R$，它包含了所有不包含自身的集合。即 $R = \{S \mid S \text{ 是一個集合且 } S \notin S\}$。
    *   **問題：** $R$ 是否包含自身？
        *   **情況一：** 如果 $R \in R$，根據 $R$ 的定義，這意味著 $R$ 不包含自身 (即 $R \notin R$)。這導致矛盾。
        *   **情況二：** 如果 $R \notin R$，根據 $R$ 的定義，這意味著 $R$ 應該包含自身 (即 $R \in R$)。這也導致矛盾。
*   **與相鄰概念的關聯：** 羅素悖論表明，我們不能隨意定義集合，否則會導致邏輯上的矛盾。這推動了數學基礎學的發展，強調了嚴謹的公理化方法對於構建一致數學體系的重要性。對於初學者，只需了解其概念，無需深入研究公理化集合論。

### 5. 常見錯誤與澄清

#### 元素與子集合的混淆

*   **錯誤範例：**
    *   認為 $a \subseteq \{a, b\}$。
    *   認為 $\{a\} \in \{a, b\}$。
*   **澄清：**
    *   $a$ 是一個元素，而 $\{a\}$ 是一個集合。
    *   $a \in \{a, b\}$ (元素 $a$ 屬於集合 $\{a, b\}$)。
    *   $\{a\} \subseteq \{a, b\}$ (集合 $\{a\}$ 是集合 $\{a, b\}$ 的子集合)。
    *   對於集合 $S = \{a, \{b\}, c\}$：
        *   $a \in S$ 是正確的。
        *   $\{b\} \in S$ 是正確的 (因為 $\{b\}$ 是集合 $S$ 的一個元素)。
        *   $b \notin S$ 是正確的 (因為 $b$ 不是集合 $S$ 的直接元素，而是元素 $\{b\}$ 裡面的元素)。
        *   $\{a\} \subseteq S$ 是正確的。
        *   $\{\{b\}\} \subseteq S$ 是正確的 (因為元素 $\{b\}$ 屬於 $S$，所以將其作為單一元素的集合 $\{\{b\}\}$ 是 $S$ 的子集合)。

#### 空集合的理解

*   **錯誤範例：**
    *   認為 $\emptyset = \{\emptyset\}$。
    *   認為 $|\{\emptyset\}| = 0$。
*   **澄清：**
    *   $\emptyset$ 是一個不包含任何元素的集合，所以 $|\emptyset| = 0$。
    *   $\{\emptyset\}$ 是一個包含「空集合」作為單一元素的集合，所以 $|\{\emptyset\}| = 1$。它不是空集合。
    *   $\{0\}$ 是一個包含數字 $0$ 作為單一元素的集合，其基數為 $1$。它也不等於空集合。

#### 真子集與子集的區別

*   **錯誤範例：**
    *   認為 $A \subset A$ 恆成立。
*   **澄清：**
    *   $A \subseteq B$ 意味著 $A$ 的所有元素都在 $B$ 中。這允許 $A$ 和 $B$ 完全相等。
    *   $A \subset B$ (真子集) 意味著 $A \subseteq B$ **並且** $A \neq B$。也就是說，$B$ 必須至少包含一個不在 $A$ 中的元素。
    *   因此，$A \subseteq A$ 永遠成立，但 $A \subset A$ **永遠不成立** (因為 $A$ 不可能真包含於它自己)。

### 6. 小練習（附詳解）

#### 小練習 1：集合表示法與基本概念

設集合 $A = \{x \in \mathbb{Z} \mid -3 < x \le 2 \}$。

1.  用列舉法表示集合 $A$。
2.  求集合 $A$ 的基數 $|A|$。
3.  判斷以下敘述是否正確：
    a) $0 \in A$
    b) $-3 \in A$
    c) $2 \subseteq A$
    d) $\{2\} \subseteq A$

**詳解：**

1.  **用列舉法表示集合 $A$：**
    *   集合 $A$ 包含所有大於 -3 且小於等於 2 的整數。
    *   這些整數是：-2, -1, 0, 1, 2。
    *   因此，$A = \{-2, -1, 0, 1, 2\}$。

2.  **求集合 $A$ 的基數 $|A|$：**
    *   集合 $A$ 包含 5 個元素。
    *   因此，$|A| = 5$。

3.  **判斷以下敘述是否正確：**
    a) **$0 \in A$**
        *   由於 $0$ 是介於 -3 和 2 之間的整數，且 $0$ 包含在集合 $A$ 中。
        *   **正確。**
    b) **$-3 \in A$**
        *   集合 $A$ 的定義是 $x > -3$，所以 -3 不在集合 $A$ 中。
        *   **不正確。**
    c) **$2 \subseteq A$**
        *   $2$ 是一個元素，而不是一個集合。子集關係 $\subseteq$ 只能用於兩個集合之間。
        *   **不正確。**
    d) **$\{2\} \subseteq A$**
        *   $\{2\}$ 是一個以 $2$ 為唯一元素的集合。由於 $2 \in A$，所以集合 $\{2\}$ 是 $A$ 的子集合。
        *   **正確。**

-----

#### 小練習 2：子集合與冪集合

設集合 $B = \{p, \{q\}\}$。

1.  列出集合 $B$ 的所有子集合。
2.  求集合 $B$ 的冪集合 $P(B)$ 的基數 $|P(B)|$。
3.  判斷以下敘述是否正確：
    a) $\{q\} \in B$
    b) $q \in B$
    c) $\{p\} \subseteq B$
    d) $\{\{q\}\} \subseteq B$
    e) $B \subseteq P(B)$

**詳解：**

1.  **列出集合 $B$ 的所有子集合：**
    *   集合 $B$ 有兩個元素：$p$ 和 $\{q\}$。
    *   空集合：$\emptyset$
    *   包含一個元素的集合：$\{p\}$, $\{\{q\}\}$
    *   包含兩個元素的集合：$\{p, \{q\}\}$ (這就是 $B$ 本身)
    *   所以，$B$ 的所有子集合是 $\{\emptyset, \{p\}, \{\{q\}\}, \{p, \{q\}\}\}$。

2.  **求集合 $B$ 的冪集合 $P(B)$ 的基數 $|P(B)|$：**
    *   集合 $B$ 有 2 個元素，即 $|B|=2$。
    *   冪集合的基數為 $2^{|B|}$。
    *   所以，$|P(B)| = 2^2 = 4$。

3.  **判斷以下敘述是否正確：**
    a) **$\{q\} \in B$**
        *   集合 $B = \{p, \{q\}\}$ 的第二個元素確實是 $\{q\}$。
        *   **正確。**
    b) **$q \in B$**
        *   $q$ 本身不是集合 $B$ 的直接元素。元素是 $p$ 和 $\{q\}$。
        *   **不正確。**
    c) **$\{p\} \subseteq B$**
        *   由於 $p \in B$，所以以 $p$ 為單一元素的集合 $\{p\}$ 是 $B$ 的子集合。
        *   **正確。**
    d) **$\{\{q\}\} \subseteq B$**
        *   由於 $\{q\} \in B$，所以以 $\{q\}$ 為單一元素的集合 $\{\{q\}\}$ 是 $B$ 的子集合。
        *   **正確。**
    e) **$B \subseteq P(B)$**
        *   $B = \{p, \{q\}\}$。
        *   $P(B) = \{\emptyset, \{p\}, \{\{q\}\}, \{p, \{q\}\}\}$。
        *   要使 $B \subseteq P(B)$，則 $B$ 的所有元素都必須是 $P(B)$ 的元素。
        *   元素 $p$ 是一個原子元素，它不是 $P(B)$ 中的任何一個集合元素。
        *   元素 $\{q\}$ 是一個集合，它也不是 $P(B)$ 中的任何一個集合元素 (儘管 $\{q\}$ 是 $B$ 的一個元素，但它不是 $P(B)$ 的元素。注意 $P(B)$ 中的元素都是 *集合*)。
        *   **不正確。** (通常情況下，一個集合不會是其冪集合的子集合，除非該集合本身是一個包含其所有元素的集合的集合，這在一般的語境中不成立。)
