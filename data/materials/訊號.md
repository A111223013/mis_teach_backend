### 訊號：資訊的載體與分析基礎

#### 什麼是訊號？

**定義/核心觀念**
在工程與科學領域，**訊號（Signal）**是指一種攜帶資訊的物理量或現象。它通常是時間、空間或其他獨立變數的函數。訊號可以是任何能被量測、記錄或傳輸的資訊載體，例如聲音、影像、電壓、電流、溫度等。訊號的變化模式包含了可供解讀的資訊。

**例子**
*   **聲音訊號：** 人說話時空氣壓力的變化，可用麥克風轉換為電壓訊號。這個電壓訊號隨時間變化，攜帶了語音內容。
*   **影像訊號：** 電視或手機螢幕上顯示的亮度與色彩資訊，通常以數位形式儲存和處理。每個像素的顏色值就是一個訊號。
*   **心電圖（ECG）訊號：** 量測心臟活動產生的電位變化，反映心臟的健康狀況。醫生透過分析訊號波形來診斷疾病。
*   **股票價格訊號：** 股市中股票價格隨時間的變化，包含市場資訊，可以被記錄和分析以預測趨勢。

**與相鄰概念的關聯**
訊號是**系統（System）**的輸入和輸出。系統對訊號進行處理、轉換或傳輸，以達成特定目的。例如，一個音響放大器就是一個系統，它接收微弱的音頻訊號（輸入），將其放大後輸出到揚聲器。**訊號處理（Signal Processing）**是一門研究如何分析、操作和修改訊號的學科。

#### 訊號的分類

**定義/核心觀念**
訊號可依據其性質和表示方式進行多種分類。了解這些分類有助於選擇合適的分析和處理方法。

##### 連續時間訊號與離散時間訊號

**定義/核心觀念**
此分類基於訊號的**獨立變數**（通常是時間）是否連續。
*   **連續時間訊號（Continuous-Time Signal, CT）：** 訊號在所有時間點（或連續的獨立變數範圍內）都有定義。其獨立變數是連續的，可用實數 $t$ 表示。
    *   數學表示：$x(t)$。
*   **離散時間訊號（Discrete-Time Signal, DT）：** 訊號僅在特定的離散時間點（或離散的獨立變數值）上定義。其獨立變數是離散的，通常用整數 $n$ 表示。
    *   數學表示：$x[n]$。

**例子**
*   **連續時間訊號：** 聲波、溫度計的讀數、電池的電壓。它們在任何時刻都有一個確定的值。
*   **離散時間訊號：** 股市每日收盤價、CD 上的數位音訊（由連續音訊取樣而來）、每小時的氣溫記錄。

**與相鄰概念的關聯**
連續時間訊號經過**取樣（Sampling）**後可以轉換為離散時間訊號。取樣是數位訊號處理的基礎。

##### 類比訊號與數位訊號

**定義/核心觀念**
此分類基於訊號的**值域（振幅）**是否連續。
*   **類比訊號（Analog Signal）：** 訊號的值域是連續的，可以在任何範圍內取任意值。它通常與物理量直接相關。
*   **數位訊號（Digital Signal）：** 訊號的值域是離散的，只能取有限個預定義的值。它通常由類比訊號經過取樣和**量化（Quantization）**後產生。

**例子**
*   **類比訊號：** 麥克風捕捉到的原始聲波電壓、傳統電話線路中的語音訊號。這些訊號的電壓值可以連續變化。
*   **數位訊號：** 電腦中的二進位資料（0和1）、CD 或 MP3 播放器中的音訊、數位相機拍攝的影像。這些訊號的值只能是有限個離散的整數。

**與相鄰概念的關聯**
**類比數位轉換器（Analog-to-Digital Converter, ADC）**將類比訊號轉換為數位訊號，而**數位類比轉換器（Digital-to-Analog Converter, DAC）**則執行逆向操作。這是現代電子設備的核心技術。

##### 確定性訊號與隨機訊號

**定義/核心觀念**
此分類基於訊號的**可預測性**。
*   **確定性訊號（Deterministic Signal）：** 訊號的未來行為可以被精確地預測或根據其數學模型完全確定。
*   **隨機訊號（Random Signal / Stochastic Signal）：** 訊號的未來行為無法被精確預測，只能用機率和統計方法來描述。通常含有雜訊（Noise）。

**例子**
*   **確定性訊號：** 正弦波 $x(t) = A \sin(\omega t + \phi)$、單位步階訊號。這些訊號的任何時刻值都可以精確計算。
*   **隨機訊號：** 來自天線的宇宙背景輻射、熱雜訊、語音通訊中的背景雜訊。這些訊號無法寫出精確的數學表達式，只能用其統計特性（如平均值、方差）來描述。

**與相鄰概念的關聯**
隨機訊號的分析通常涉及**機率論（Probability Theory）**和**隨機過程（Stochastic Process）**的工具。

##### 能量訊號與功率訊號

**定義/核心觀念**
此分類基於訊號的總能量和平均功率是否有限。

*   **訊號的總能量 $E_x$：**
    *   對於連續時間訊號 $x(t)$：$E_x = \int_{-\infty}^{\infty} |x(t)|^2 dt$
    *   對於離散時間訊號 $x[n]$：$E_x = \sum_{n=-\infty}^{\infty} |x[n]|^2$
*   **訊號的平均功率 $P_x$：**
    *   對於連續時間訊號 $x(t)$：$P_x = \lim_{T \to \infty} \frac{1}{2T} \int_{-T}^{T} |x(t)|^2 dt$
    *   對於離散時間訊號 $x[n]$：$P_x = \lim_{N \to \infty} \frac{1}{2N+1} \sum_{n=-N}^{N} |x[n]|^2$

*   **能量訊號（Energy Signal）：** 總能量 $E_x$ 是有限且非零的 ($0 < E_x < \infty$)。對於能量訊號，平均功率 $P_x = 0$。
*   **功率訊號（Power Signal）：** 平均功率 $P_x$ 是有限且非零的 ($0 < P_x < \infty$)。對於功率訊號，總能量 $E_x = \infty$。

**例子**
*   **能量訊號：** 僅在有限時間內存在的訊號，如單一脈衝訊號、隨時間衰減的指數訊號 $e^{-at} u(t)$ ($a>0$)。這些訊號最終會衰減到零。
*   **功率訊號：** 存在於無限時間內的週期訊號，如正弦波、單位步階訊號 $u(t)$。這些訊號的振幅不會衰減到零，因此總能量無限。

**與相鄰概念的關聯**
能量和功率是訊號在分析和設計系統時的重要指標，尤其是在通訊和訊號處理中評估訊號強度和效率。

#### 基本訊號運算

**定義/核心觀念**
對訊號進行基本的時間域（Time Domain）運算，是理解訊號行為和設計系統的基礎。這些運算包括時間平移、時間縮放和時間反轉。

##### 時間平移（Time Shifting）

**核心觀念**
將訊號在時間軸上向左或向右移動。
*   $x(t - t_0)$ 或 $x[n - n_0]$：訊號向右（延遲）平移 $t_0$（或 $n_0$）個單位。
*   $x(t + t_0)$ 或 $x[n + n_0]$：訊號向左（提前）平移 $t_0$（或 $n_0$）個單位。

**例子**
若 $x(t)$ 是一個在 $t=0$ 處有峰值的脈衝訊號，則 $x(t-2)$ 會在 $t=2$ 處有相同的峰值。

##### 時間縮放（Time Scaling）

**核心觀念**
改變訊號在時間軸上的“速度”，使訊號變寬或變窄。
*   $x(at)$ 或 $x[an]$：當 $a > 1$ 時，訊號被壓縮（事件發生得更快）。
*   $x(at)$ 或 $x[an]$：當 $0 < a < 1$ 時，訊號被擴展（事件發生得更慢）。

**例子**
若 $x(t)$ 是一個持續 1 秒的方波，則 $x(2t)$ 是一個持續 0.5 秒的方波，而 $x(0.5t)$ 是一個持續 2 秒的方波。

##### 時間反轉（Time Reversal / Flipping）

**核心觀念**
將訊號沿著垂直軸（$t=0$ 軸）翻轉。
*   $x(-t)$ 或 $x[-n]$：訊號在時間軸上反轉。

**例子**
若 $x(t)$ 是一個從 $t=0$ 到 $t=1$ 遞增的斜坡訊號，則 $x(-t)$ 是一個從 $t=-1$ 到 $t=0$ 遞減的斜坡訊號。

**與相鄰概念的關聯**
這些基本運算是分析訊號與系統行為（如線性時不變系統的響應）的基礎，也是傅立葉轉換和拉普拉斯轉換等頻域分析的性質來源。

### 典型訊號範例與應用

#### 常用基本訊號

**定義/核心觀念**
有一些特殊的訊號在訊號處理中扮演基礎角色，它們可以用來建構或分析更複雜的訊號。

##### 單位脈衝訊號（Unit Impulse Signal / Dirac Delta Function）

**定義/核心觀念**
*   **連續時間：$\delta(t)$。** 在 $t=0$ 時值為無限大，但在其他所有時間點都為零，且其總面積為 1。
    *   $\delta(t) = 0$ for $t \neq 0$
    *   $\int_{-\infty}^{\infty} \delta(t) dt = 1$
*   **離散時間：$\delta[n]$。** 在 $n=0$ 時值為 1，在其他所有點都為零。
    *   $\delta[n] = 1$ for $n = 0$
    *   $\delta[n] = 0$ for $n \neq 0$

**例子**
在物理上，一個極短時間內發生的極強事件（如閃電或敲擊）可以用脈衝訊號建模。在數學上，它可以視為一個極窄的矩形脈衝當寬度趨近於零時的極限。

**與相鄰概念的關聯**
單位脈衝訊號是**線性時不變（LTI）系統**分析的基石。系統對單位脈衝訊號的響應稱為**脈衝響應（Impulse Response）**，它完全描述了LTI系統的特性。任何訊號都可以表示為無數個加權和時間平移的脈衝訊號的疊加（稱為摺積）。

##### 單位步階訊號（Unit Step Signal）

**定義/核心觀念**
*   **連續時間：$u(t)$。** 在 $t \ge 0$ 時值為 1，在 $t < 0$ 時值為 0。
    *   $u(t) = \begin{cases} 1 & t \ge 0 \\ 0 & t < 0 \end{cases}$
*   **離散時間：$u[n]$。** 在 $n \ge 0$ 時值為 1，在 $n < 0$ 時值為 0。
    *   $u[n] = \begin{cases} 1 & n \ge 0 \\ 0 & n < 0 \end{cases}$

**例子**
當我們打開電燈開關，電壓從 0 瞬間變為某個固定值，這可以近似為單位步階訊號。它常被用來表示訊號的開始。

**與相鄰概念的關聯**
單位步階訊號是分析系統對突然輸入響應的基礎。它與單位脈衝訊號密切相關：$\delta(t) = \frac{du(t)}{dt}$，反之，$u(t) = \int_{-\infty}^{t} \delta(\tau) d\tau$。

##### 實指數訊號（Real Exponential Signal）

**定義/核心觀念**
*   **連續時間：** $x(t) = Ae^{\alpha t}$
*   **離散時間：** $x[n] = A\alpha^n$

*   如果 $\alpha > 0$（連續時間）或 $|\alpha| > 1$（離散時間），訊號隨時間增長。
*   如果 $\alpha < 0$（連續時間）或 $0 < |\alpha| < 1$（離散時間），訊號隨時間衰減。
*   如果 $\alpha = 0$（連續時間）或 $|\alpha| = 1$（離散時間），訊號為常數或振盪。

**例子**
*   **衰減指數訊號：** $e^{-at}u(t)$，描述電路中電容放電、放射性衰變等現象。
*   **增長指數訊號：** $e^{at}u(t)$，描述複利增長、細菌繁殖等現象。

**與相鄰概念的關聯**
指數訊號是許多線性微分方程和差分方程的特徵函數，對於分析LTI系統的頻率響應和穩定性至關重要。它們是拉普拉斯轉換和Z轉換的基礎。

##### 弦波訊號（Sinusoidal Signal）

**定義/核心觀念**
*   **連續時間：** $x(t) = A \cos(\omega_0 t + \phi)$ 或 $A \sin(\omega_0 t + \phi)$
    其中 $A$ 是振幅，$\omega_0$ 是角頻率（rad/s），$\phi$ 是相位角。
*   **離散時間：** $x[n] = A \cos(\Omega_0 n + \phi)$ 或 $A \sin(\Omega_0 n + \phi)$
    其中 $\Omega_0$ 是離散時間角頻率（rad/sample）。

**例子**
交流電的電壓或電流、音樂聲波、無線電訊號。

**與相鄰概念的關聯**
弦波訊號是週期訊號的基本組成部分。**傅立葉級數（Fourier Series）**指出任何週期訊號都可以表示為一系列弦波訊號的加權和。**傅立葉轉換（Fourier Transform）**則將非週期訊號分解為連續頻譜的弦波成分。

### 與相鄰概念的關聯

#### 訊號與系統

**核心觀念**
訊號與系統是訊號處理領域的兩大基石。訊號是承載資訊的載體，而**系統（System）**則是對訊號進行操作、轉換或處理的實體。一個系統可以是一個物理裝置（如濾波器、放大器），也可以是一個數學模型（如演算法）。

**例子**
*   **音響系統：** 麥克風將聲壓訊號（輸入）轉換為電壓訊號，放大器放大電壓訊號，揚聲器將放大的電壓訊號轉換回聲壓訊號（輸出）。
*   **通訊系統：** 發射端將原始訊號調變、編碼後傳輸，接收端將接收到的訊號解調、解碼以還原原始訊號。

**關聯**
*   **輸入/輸出：** 訊號是系統的輸入，經過系統處理後產生輸出訊號。
*   **系統特性：** 系統的特性（如線性、時不變性、因果性、穩定性）決定了它如何響應不同的輸入訊號。例如，**線性時不變（LTI）系統**是訊號處理中研究最深入的系統類型，其行為可完全由脈衝響應描述。

#### 訊號與頻域分析

**核心觀念**
許多訊號的特性在時間域中不易觀察，但在頻域中則清晰可見。**頻域分析（Frequency Domain Analysis）**將訊號從其在時間上的變化，轉換為其包含的各種頻率成分。

**核心工具**
*   **傅立葉級數（Fourier Series）：** 用於分析週期性連續時間訊號，將其分解為一系列基本頻率（基頻及其諧波）的弦波和。
*   **傅立葉轉換（Fourier Transform）：** 用於分析非週期性連續時間訊號，將其轉換為連續頻譜，揭示訊號中包含的所有頻率成分。
    *   $X(j\omega) = \int_{-\infty}^{\infty} x(t) e^{-j\omega t} dt$
*   **離散時間傅立葉轉換（Discrete-Time Fourier Transform, DTFT）：** 用於分析非週期性離散時間訊號。
    *   $X(e^{j\Omega}) = \sum_{n=-\infty}^{\infty} x[n] e^{-j\Omega n}$
*   **離散傅立葉轉換（Discrete Fourier Transform, DFT）與快速傅立葉轉換（Fast Fourier Transform, FFT）：** 用於分析有限長度離散時間訊號的頻譜，是數位訊號處理中的核心演算法。FFT是DFT的一種高效計算方法。

**例子**
*   透過對語音訊號進行頻域分析，可以辨識說話者的音高、音色等特徵，甚至用於語音辨識。
*   分析音頻的頻譜可以判斷其是否包含高頻雜訊（如嘶嘶聲），以便設計濾波器進行處理。

**關聯**
頻域分析讓我們能從另一個角度理解訊號的本質，並設計出針對特定頻率範圍的濾波器、均衡器等訊號處理工具，實現訊號增強、雜訊抑制等功能。

#### 訊號與取樣、量化、編碼

**核心觀念**
將連續的類比訊號轉換為數位訊號的過程，是現代數位通訊和處理的基礎。這涉及取樣、量化和編碼三個關鍵步驟。

##### 取樣（Sampling）

**核心觀念**
將連續時間訊號在離散的時間點上進行測量，從而獲得離散時間訊號。
*   **取樣頻率 $f_s$：** 每秒取樣的次數。
*   **奈奎斯特-香農取樣定理（Nyquist-Shannon Sampling Theorem）：** 若要無失真地從取樣訊號重建原始連續時間訊號，取樣頻率 $f_s$ 必須至少是訊號最高頻率成分 $f_{max}$ 的兩倍 ($f_s \ge 2f_{max}$)。否則會發生**混疊（Aliasing）**現象。

##### 量化（Quantization）

**核心觀念**
將取樣後訊號的連續振幅值映射到有限個離散的振幅層次。這個過程會引入**量化雜訊（Quantization Noise）**或量化誤差。
*   **位元深度（Bit Depth）：** 用於表示每個取樣值離散層次的位元數。位元深度越高（例如從8位元到16位元），可表示的層次越多，量化誤差越小。

##### 編碼（Encoding）

**核心觀念**
將量化後的離散值轉換為二進位碼（或其他數位碼），以便於儲存、傳輸和處理。這是將抽象的數位值轉化為實際的電氣或光學脈衝。

**例子**
當你使用麥克風錄製聲音，麥克風產生的是類比電壓訊號。
1.  **取樣：** 聲卡以數萬赫茲的頻率對電壓進行取樣，得到離散時間電壓值。
2.  **量化：** 每個取樣值被轉換為預設的數位層次之一（例如，16位元音訊有 $2^{16} = 65536$ 個層次）。
3.  **編碼：** 這些量化後的數值被轉換為二進位資料（例如，`01101011...`），形成最終的數位音訊檔案。

**關聯**
這三個步驟共同完成了類比數位轉換（ADC），是將物理世界中的類比資訊引入數位世界的橋樑。數位訊號相較於類比訊號具有抗雜訊能力強、易於儲存和處理的優點，是現代通訊和計算的基礎。

### 常見錯誤與澄清

#### 1. 連續時間訊號與類比訊號、離散時間訊號與數位訊號的混淆

**常見錯誤**
很多人會將「連續時間」與「類比」視為同義詞，將「離散時間」與「數位」視為同義詞。這導致理解上的偏差。

**澄清**
「連續/離散時間」和「類比/數位」是兩個獨立的分類維度，分別描述訊號的**獨立變數（時間軸）**和**值域（振幅）**的特性。

*   **時間軸（獨立變數）的連續/離散：**
    *   **連續時間訊號（CT）：** 訊號在時間軸上連續有定義，如 $x(t)$。
    *   **離散時間訊號（DT）：** 訊號只在離散時間點有定義，如 $x[n]$。
*   **值域（振幅）的連續/離散：**
    *   **類比訊號：** 訊號的振幅值可以是連續的任意值。
    *   **數位訊號：** 訊號的振幅值是離散的、有限的幾個預定義值。

**組合可能性**
|           | 值域連續（類比）                                     | 值域離散（數位）                                     |
| :-------- | :--------------------------------------------------- | :--------------------------------------------------- |
| **時間連續（CT）** | **連續時間類比訊號** (例如：原始聲波、電壓、溫度變化)      | **連續時間數位訊號** (較少見，理論上可能。例如，一個時間上連續但振幅只能取0或1的訊號) |
| **時間離散（DT）** | **離散時間類比訊號** (例如：取樣但未量化的訊號，每個取樣點仍可取連續振幅) | **離散時間數位訊號** (例如：電腦中的音訊、影像資料。這是在數位訊號處理中最常見的訊號形式) |

通常我們在談論數位訊號處理時，遇到的多是**離散時間數位訊號**，這是因為經過取樣（變為離散時間）和量化（變為數位）之後的結果。

#### 2. 能量訊號與功率訊號的判斷錯誤

**常見錯誤**
誤以為所有訊號都是能量訊號或功率訊號。或者在計算能量或功率時，未考慮訊號的持續時間。

**澄清**
一個訊號要麼是能量訊號，要麼是功率訊號，但不能兩者都是（除非訊號恆為零，則能量和功率都為零）。存在既非能量訊號也非功率訊號的訊號（例如 $x(t) = t \cdot u(t)$，其能量和功率都趨於無限）。

*   **能量訊號的特點：**
    *   通常只在有限時間範圍內存在，或隨時間快速衰減以致其總能量有限。
    *   其總能量 $E_x$ 有限且非零，但平均功率 $P_x$ 為零。
*   **功率訊號的特點：**
    *   通常存在於無限時間範圍內，且振幅不會衰減到零。
    *   其總能量 $E_x$ 無限，但平均功率 $P_x$ 有限且非零。
*   **判斷關鍵：**
    *   對於週期訊號，直接計算其平均功率（一個週期內的能量除以週期長度）會是有限值，故為功率訊號。
    *   對於有限持續時間的非零訊號，其總能量必然有限，故為能量訊號。
    *   對於無限持續時間的非週期訊號，需根據定義積分或求和，判斷能量和功率哪個有限。

#### 3. 時間縮放 $x(at)$ 的理解偏差

**常見錯誤**
在 $x(at)$ 中，當 $a > 1$ 時，訊號是「加速」或「壓縮」；當 $0 < a < 1$ 時，訊號是「減速」或「擴展」。有時會將 $a$ 的作用方向搞錯。

**澄清**
考慮一個在 $t=T_0$ 時刻發生某事件的訊號 $x(t)$。對於 $x(at)$，這個事件會發生在 $at=T_0$，即 $t = T_0/a$ 時刻。

*   如果 $a > 1$，例如 $a=2$，則 $t = T_0/2 < T_0$。事件發生得更早，訊號在時間軸上被壓縮。
*   如果 $0 < a < 1$，例如 $a=0.5$，則 $t = T_0/0.5 = 2T_0 > T_0$。事件發生得更晚，訊號在時間軸上被擴展。

**記憶技巧**
把 $a$ 想成是時間流逝的速度因子。 $a=2$ 表示時間過得比原來快兩倍，訊號的事件在原來一半的時間就發生完了，所以訊號看起來被壓縮了。 $a=0.5$ 表示時間過得比原來慢兩倍，訊號的事件需要原來兩倍的時間才能發生完，所以訊號看起來被擴展了。

### 小練習（附詳解）

#### 小練習 1：訊號分類與運算

給定一個連續時間訊號 $x(t) = 4\cos(2\pi t)u(t-1)$。
請回答以下問題：
1.  該訊號是連續時間訊號還是離散時間訊號？
2.  該訊號是類比訊號還是數位訊號？
3.  該訊號是確定性訊號還是隨機訊號？
4.  計算並繪製 $y(t) = x(2t+2)$ 的波形示意圖（可口頭描述或草繪）。

**詳解**

1.  **該訊號是連續時間訊號還是離散時間訊號？**
    *   訊號 $x(t)$ 的獨立變數是 $t$，它是一個連續變數。因此，該訊號是**連續時間訊號**。

2.  **該訊號是類比訊號還是數位訊號？**
    *   訊號 $x(t)$ 的值域 $4\cos(2\pi t)$ 可以在 $-4$ 到 $4$ 之間取任意實數值。因此，該訊號是**類比訊號**。

3.  **該訊號是確定性訊號還是隨機訊號？**
    *   訊號 $x(t)$ 的數學表達式是完全已知的，其未來行為可以被精確預測。因此，該訊號是**確定性訊號**。

4.  **計算並繪製 $y(t) = x(2t+2)$ 的波形示意圖。**
    *   **步驟一：替換變數**
        將 $x(t)$ 中的 $t$ 替換為 $(2t+2)$：
        $y(t) = 4\cos(2\pi (2t+2))u((2t+2)-1)$
    *   **步驟二：化簡三角函數**
        $y(t) = 4\cos(4\pi t + 4\pi)u(2t+1)$
        由於 $\cos(\theta + 2k\pi) = \cos(\theta)$，所以 $\cos(4\pi t + 4\pi) = \cos(4\pi t)$。
        $y(t) = 4\cos(4\pi t)u(2t+1)$
    *   **步驟三：分析單位步階訊號**
        單位步階訊號 $u(2t+1)$ 在 $2t+1 \ge 0$ 時為 1，即 $2t \ge -1 \implies t \ge -0.5$ 時為 1；否則為 0。
    *   **步驟四：最終表達式**
        $y(t) = \begin{cases} 4\cos(4\pi t) & t \ge -0.5 \\ 0 & t < -0.5 \end{cases}$
    *   **步驟五：波形描述**
        原始訊號 $x(t)$ 是一個從 $t=1$ 開始的正弦波（頻率為 $1$ Hz，振幅為 $4$）。
        新訊號 $y(t)$ 是一個從 $t=-0.5$ 開始的正弦波：
        *   **起始點：** 從 $t=-0.5$ 而非 $t=1$ 開始。
        *   **頻率：** 原始角頻率是 $2\pi$，新的角頻率是 $4\pi$。這表示頻率變為 $2$ Hz，訊號被壓縮（比原來快兩倍）。
        *   **振幅：** 振幅仍為 $4$。

        因此，$y(t)$ 是一個從 $t=-0.5$ 開始的，振幅為 4，頻率為 2 Hz 的餘弦波。它相對於 $x(t)$ 被向左平移（提前）且被壓縮。

#### 小練習 2：能量訊號與功率訊號判斷

請判斷以下訊號是能量訊號還是功率訊號，並計算其能量或平均功率。

1.  $x_1(t) = e^{-3t}u(t)$
2.  $x_2(t) = 5\cos(2t)$

**詳解**

1.  **$x_1(t) = e^{-3t}u(t)$**
    *   **步驟一：判斷類型**
        該訊號在 $t \ge 0$ 時為 $e^{-3t}$，且隨時間指數衰減；在 $t < 0$ 時為 0。由於訊號最終會衰減到零，預期其總能量是有限的。
    *   **步驟二：計算能量 $E_{x_1}$**
        $E_{x_1} = \int_{-\infty}^{\infty} |x_1(t)|^2 dt$
        因為 $u(t)$ 在 $t<0$ 時為 0，所以積分範圍為 $[0, \infty)$：
        $E_{x_1} = \int_{0}^{\infty} (e^{-3t})^2 dt = \int_{0}^{\infty} e^{-6t} dt$
        $E_{x_1} = \left[ -\frac{1}{6} e^{-6t} \right]_{0}^{\infty}$
        $E_{x_1} = \left( -\frac{1}{6} \lim_{t \to \infty} e^{-6t} \right) - \left( -\frac{1}{6} e^{0} \right)$
        $E_{x_1} = \left( -\frac{1}{6} \cdot 0 \right) - \left( -\frac{1}{6} \cdot 1 \right) = 0 - (-\frac{1}{6}) = \frac{1}{6}$
    *   **步驟三：結論**
        由於 $0 < E_{x_1} = \frac{1}{6} < \infty$，訊號 $x_1(t)$ 是一個**能量訊號**。

2.  **$x_2(t) = 5\cos(2t)$**
    *   **步驟一：判斷類型**
        該訊號是一個無限持續的週期訊號，振幅不衰減。預期其總能量是無限的，但平均功率是有限的。
    *   **步驟二：計算平均功率 $P_{x_2}$**
        對於週期訊號，平均功率可以在一個週期內計算：$P_x = \frac{1}{T_0} \int_{T_0} |x(t)|^2 dt$。
        訊號 $x_2(t) = 5\cos(2t)$ 的角頻率 $\omega_0 = 2$ rad/s，所以週期 $T_0 = \frac{2\pi}{\omega_0} = \frac{2\pi}{2} = \pi$ 秒。
        $P_{x_2} = \frac{1}{\pi} \int_{0}^{\pi} |5\cos(2t)|^2 dt$
        $P_{x_2} = \frac{1}{\pi} \int_{0}^{\pi} 25\cos^2(2t) dt$
    *   **步驟三：使用三角恆等式 $\cos^2(\theta) = \frac{1 + \cos(2\theta)}{2}$**
        $P_{x_2} = \frac{25}{\pi} \int_{0}^{\pi} \frac{1 + \cos(4t)}{2} dt$
        $P_{x_2} = \frac{25}{2\pi} \int_{0}^{\pi} (1 + \cos(4t)) dt$
        $P_{x_2} = \frac{25}{2\pi} \left[ t + \frac{\sin(4t)}{4} \right]_{0}^{\pi}$
        $P_{x_2} = \frac{25}{2\pi} \left[ (\pi + \frac{\sin(4\pi)}{4}) - (0 + \frac{\sin(0)}{4}) \right]$
        $P_{x_2} = \frac{25}{2\pi} (\pi + 0 - 0 - 0) = \frac{25}{2}$
    *   **步驟四：結論**
        由於 $0 < P_{x_2} = \frac{25}{2} < \infty$，訊號 $x_2(t)$ 是一個**功率訊號**。
