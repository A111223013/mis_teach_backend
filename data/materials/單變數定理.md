# 單變數定理：均值定理 (Mean Value Theorem)

### 核心概念與定義

#### 什麼是單變數定理？
單變數定理是指在一個僅包含一個自變數的函數上成立的數學定理。這些定理構成了微積分學的基石，它們闡明了函數的連續性、可微分性、單調性以及積分等核心性質之間的關係。在這些定理中，均值定理 (Mean Value Theorem, MVT) 及其前身羅爾定理 (Rolle's Theorem) 扮演著至關重要的角色，它們將函數的瞬時變化率（導數）與其在區間上的平均變化率聯繫起來。

-----

#### 羅爾定理 (Rolle's Theorem)

##### 定義
如果一個函數 $f(x)$ 滿足以下三個條件：
1.  在閉區間 $[a, b]$ 上連續。
2.  在開區間 $(a, b)$ 上可微分。
3.  在區間端點處函數值相等，即 $f(a) = f(b)$。
那麼，在開區間 $(a, b)$ 內至少存在一個點 $c$，使得 $f'(c) = 0$。

##### 核心觀念
羅爾定理的關鍵在於，如果一個連續可微分的函數在區間兩端的函數值相同，那麼在該區間內部，函數圖形至少會有一個「水平切線」，亦即導數為零的點。這代表函數在此點處達到局部最大值或局部最小值。

##### 幾何解釋
想像一個連續平滑的曲線（代表函數 $f(x)$）。如果這個曲線的起點 $(a, f(a))$ 和終點 $(b, f(b))$ 位於相同的高度（即 $f(a) = f(b)$），那麼在曲線從起點走到終點的過程中，它一定會經歷至少一個「轉折點」，在這個轉折點上，曲線的切線是水平的，斜率為零。

-----

#### 均值定理 (Mean Value Theorem, MVT)

##### 定義
如果一個函數 $f(x)$ 滿足以下兩個條件：
1.  在閉區間 $[a, b]$ 上連續。
2.  在開區間 $(a, b)$ 上可微分。
那麼，在開區間 $(a, b)$ 內至少存在一個點 $c$，使得：
$$f'(c) = \frac{f(b) - f(a)}{b - a}$$

##### 核心觀念
均值定理指出，一個連續可微分函數在一個區間上的平均變化率（即割線斜率）等於在該區間內部某一點的瞬時變化率（即切線斜率）。

##### 幾何解釋
想像一個連續平滑的曲線。連接曲線上的兩點 $(a, f(a))$ 和 $(b, f(b))$ 的直線稱為割線，其斜率為 $\frac{f(b) - f(a)}{b - a}$。均值定理告訴我們，在曲線的這兩點之間，至少存在一個點 $c$，使得在點 $(c, f(c))$ 處的切線平行於這條割線。換句話說，切線的斜率 $f'(c)$ 等於割線的斜率。

##### 與羅爾定理的關聯
羅爾定理是均值定理的一個特殊情況。
若在均值定理的條件下，額外加上 $f(a) = f(b)$，則均值定理的結論會變成：
$$f'(c) = \frac{f(b) - f(a)}{b - a} = \frac{f(a) - f(a)}{b - a} = \frac{0}{b - a} = 0$$
這正是羅爾定理的結論。因此，羅爾定理可以看作是均值定理在割線斜率為零時的特例。

### 典型例子與證明思路

#### 均值定理的證明思路
均值定理的證明通常是通過構造一個輔助函數，然後對這個輔助函數應用羅爾定理來完成。

**證明步驟：**
1.  **定義割線函數 $L(x)$：**
    考慮連接點 $(a, f(a))$ 和 $(b, f(b))$ 的直線。這條割線的方程式可以寫成：
    $$L(x) = f(a) + \frac{f(b) - f(a)}{b - a}(x - a)$$
    這條直線在 $x=a$ 時 $L(a) = f(a)$，在 $x=b$ 時 $L(b) = f(b)$。

2.  **構造輔助函數 $g(x)$：**
    定義一個新的函數 $g(x)$ 為原函數 $f(x)$ 與割線函數 $L(x)$ 之差：
    $$g(x) = f(x) - L(x) = f(x) - \left[ f(a) + \frac{f(b) - f(a)}{b - a}(x - a) \right]$$

3.  **檢查 $g(x)$ 是否滿足羅爾定理的條件：**
    *   **連續性：** 由於 $f(x)$ 在 $[a, b]$ 上連續，$L(x)$ 是一個線性函數（多項式），因此也在 $[a, b]$ 上連續。兩個連續函數的差 $g(x)$ 也在 $[a, b]$ 上連續。
    *   **可微分性：** 由於 $f(x)$ 在 $(a, b)$ 上可微分，$L(x)$ 是一個線性函數，因此也在 $(a, b)$ 上可微分。兩個可微分函數的差 $g(x)$ 也在 $(a, b)$ 上可微分。
    *   **端點函數值相等：**
        $$g(a) = f(a) - \left[ f(a) + \frac{f(b) - f(a)}{b - a}(a - a) \right] = f(a) - f(a) = 0$$
        $$g(b) = f(b) - \left[ f(a) + \frac{f(b) - f(a)}{b - a}(b - a) \right] = f(b) - [f(a) + (f(b) - f(a))] = f(b) - f(b) = 0$$
        所以 $g(a) = g(b) = 0$。

4.  **應用羅爾定理：**
    由於 $g(x)$ 滿足羅爾定理的所有條件，因此在開區間 $(a, b)$ 內至少存在一個點 $c$，使得 $g'(c) = 0$。

5.  **求導 $g'(x)$：**
    對 $g(x) = f(x) - \left[ f(a) + \frac{f(b) - f(a)}{b - a}(x - a) \right]$ 關於 $x$ 求導：
    $$g'(x) = f'(x) - \frac{f(b) - f(a)}{b - a}$$
    （因為 $f(a)$ 和 $\frac{f(b) - f(a)}{b - a}$ 都是常數，而 $x-a$ 的導數是 $1$。）

6.  **得出結論：**
    將 $g'(c) = 0$ 代入上述導數式：
    $$f'(c) - \frac{f(b) - f(a)}{b - a} = 0$$
    $$f'(c) = \frac{f(b) - f(a)}{b - a}$$
    證畢。

#### 應用均值定理的例子

**例 1：證明函數單調性**
如果一個函數 $f(x)$ 在區間 $[a, b]$ 上連續，在 $(a, b)$ 上可微分，且對於所有的 $x \in (a, b)$ 都有 $f'(x) > 0$，證明 $f(x)$ 在 $[a, b]$ 上是嚴格遞增的。

**證明：**
任取區間 $[a, b]$ 中的兩點 $x_1, x_2$ 使得 $a \le x_1 < x_2 \le b$。
根據均值定理，在區間 $[x_1, x_2]$ 上，存在一個點 $c \in (x_1, x_2)$ 使得：
$$f'(c) = \frac{f(x_2) - f(x_1)}{x_2 - x_1}$$
因為 $x_1 < x_2$，所以 $x_2 - x_1 > 0$。
已知在開區間 $(a, b)$ 上 $f'(x) > 0$，所以 $f'(c) > 0$。
因此，$\frac{f(x_2) - f(x_1)}{x_2 - x_1} > 0$。
由於 $x_2 - x_1 > 0$，這意味著 $f(x_2) - f(x_1) > 0$，即 $f(x_2) > f(x_1)$。
所以，當 $x_1 < x_2$ 時，有 $f(x_1) < f(x_2)$，這證明了 $f(x)$ 在 $[a, b]$ 上是嚴格遞增的。

**例 2：估計函數值**
已知 $f(x) = \sqrt{x}$，估計 $\sqrt{101}$ 的值。

**解：**
我們可以考慮區間 $[100, 101]$。
$f(x) = \sqrt{x}$ 在 $[100, 101]$ 上連續，在 $(100, 101)$ 上可微分（因為 $f'(x) = \frac{1}{2\sqrt{x}}$ 在此區間內有定義）。
根據均值定理，存在 $c \in (100, 101)$ 使得：
$$f'(c) = \frac{f(101) - f(100)}{101 - 100}$$
$$\frac{1}{2\sqrt{c}} = \frac{\sqrt{101} - \sqrt{100}}{1}$$
$$\frac{1}{2\sqrt{c}} = \sqrt{101} - 10$$
由於 $c \in (100, 101)$，我們可以近似地認為 $c \approx 100$。
所以 $\frac{1}{2\sqrt{c}} \approx \frac{1}{2\sqrt{100}} = \frac{1}{2 \times 10} = \frac{1}{20} = 0.05$。
因此，$0.05 \approx \sqrt{101} - 10$。
$\sqrt{101} \approx 10 + 0.05 = 10.05$。

（實際值 $\sqrt{101} \approx 10.049875...$，這個估計非常接近。）
這個例子展示了 MVT 如何將函數的局部變化（導數）與其整體變化（區間上的差值）聯繫起來，進而用於估計。

### 與相鄰概念的關聯

#### 與中間值定理 (Intermediate Value Theorem, IVT) 的關聯
*   **中間值定理 (IVT)：** 如果一個函數 $f(x)$ 在閉區間 $[a, b]$ 上連續，且 $k$ 是介於 $f(a)$ 和 $f(b)$ 之間的任意值，那麼在開區間 $(a, b)$ 內至少存在一個點 $c$，使得 $f(c) = k$。
*   **關聯：** IVT 強調的是**連續函數**的「不跳躍」性質，保證了函數值可以取到介於端點值之間的所有數值。而 MVT 則強調**連續且可微分函數**的性質，它將函數值的變化與其導數聯繫起來。兩者都是關於函數在區間上行為的基本定理，但 MVT 引入了導數的概念，提供了更強的結論。MVT 的證明需要函數的連續性，這是 IVT 的前提，顯示了連續性在微積分中的基礎地位。

#### 與泰勒定理 (Taylor's Theorem) 的關聯
*   **泰勒定理：** 提供了用多項式逼近函數的方法，並給出了誤差（餘項）的表達式。
*   **關聯：** 均值定理可以看作是泰勒定理的一階特殊形式。
    泰勒定理帶有拉格朗日餘項的公式為：
    $$f(x) = f(a) + f'(a)(x-a) + \frac{f''(a)}{2!}(x-a)^2 + \dots + \frac{f^{(n)}(a)}{n!}(x-a)^n + R_n(x)$$
    其中餘項 $R_n(x) = \frac{f^{(n+1)}(c)}{(n+1)!}(x-a)^{n+1}$，對於某個 $c$ 介於 $a$ 和 $x$ 之間。
    當 $n=0$ 時，泰勒定理變為：
    $$f(x) = f(a) + R_0(x)$$
    $$f(x) = f(a) + \frac{f'(c)}{1!}(x-a)^1$$
    整理得到：
    $$\frac{f(x) - f(a)}{x - a} = f'(c)$$
    這正是均值定理的形式。因此，MVT 是泰勒定理最簡單（一階）的應用。

#### 與微積分基本定理 (Fundamental Theorem of Calculus, FTC) 的關聯
*   **微積分基本定理 (FTC)：** 將微分和積分聯繫起來，第一部分說明定積分的上限函數的導數是被積函數本身，第二部分說明定積分可以通過計算原函數在上下限的差值來得到。
*   **關聯：** 均值定理與 FTC 都表達了函數整體變化與局部變化之間的聯繫。
    如果我們將 MVT 應用於積分形式，會得到積分均值定理（Mean Value Theorem for Integrals）：
    如果 $f(x)$ 在 $[a, b]$ 上連續，則在 $[a, b]$ 內存在一個 $c$，使得
    $$f(c) = \frac{1}{b-a} \int_a^b f(x) dx$$
    這表示連續函數在區間上的平均值等於函數在某一點的值。
    而 FTC II ($\int_a^b f(x) dx = F(b) - F(a)$，其中 $F'(x) = f(x)$) 與 MVT ($\frac{F(b) - F(a)}{b-a} = F'(c)$) 有著形式上的相似性，都將區間端點的函數值差與某點的導數聯繫起來。MVT 關注導函數與平均變化率的關係，而 FTC 則聚焦於原函數與定積分的關係，兩者共同構成了微積分的核心。

### 進階內容：柯西均值定理 (Cauchy's Mean Value Theorem)

#### 定義
如果兩個函數 $f(x)$ 和 $g(x)$ 滿足以下條件：
1.  在閉區間 $[a, b]$ 上連續。
2.  在開區間 $(a, b)$ 上可微分。
3.  對於所有 $x \in (a, b)$，$g'(x) \neq 0$。
那麼，在開區間 $(a, b)$ 內至少存在一個點 $c$，使得：
$$\frac{f'(c)}{g'(c)} = \frac{f(b) - f(a)}{g(b) - g(a)}$$

#### 核心觀念
柯西均值定理是均值定理的一個推廣。它比較了兩個函數在同一個區間內的平均變化率之比與它們導數之比的關係。簡單來說，如果兩個函數都在同一區間內平滑變化，那麼它們的瞬時變化率之比會在某點等於它們總體變化量之比。

#### 與均值定理的關聯
當 $g(x) = x$ 時，那麼 $g'(x) = 1$，$g(b) - g(a) = b - a$。
將這些代入柯西均值定理的公式：
$$\frac{f'(c)}{1} = \frac{f(b) - f(a)}{b - a}$$
$$f'(c) = \frac{f(b) - f(a)}{b - a}$$
這正是標準的均值定理。因此，柯西均值定理是均值定理的更一般形式。

#### 與洛必達法則 (L'Hôpital's Rule) 的關係
柯西均值定理是證明洛必達法則的基礎。洛必達法則用於計算 $\frac{0}{0}$ 或 $\frac{\infty}{\infty}$ 型不定式的極限，其核心思想是通過比較分子和分母函數的導數之比來求極限。柯西均值定理恰好提供了兩個函數在區間上變化比率的聯繫，這正是洛必達法則能夠成立的數學依據。

### 常見錯誤與澄清

#### 1. 混淆定理的條件
*   **錯誤：** 在不滿足連續性或可微分性條件的函數上應用羅爾定理或均值定理。
*   **澄清：**
    *   **連續性**（閉區間 $[a, b]$ 上）：這是根本要求，如果函數有間斷點，定理不適用。例如，函數 $f(x) = \frac{1}{x}$ 在 $[-1, 1]$ 上不連續，即使 $f(-1) = f(1)$，羅爾定理也不適用。
    *   **可微分性**（開區間 $(a, b)$ 上）：這表示函數圖形必須是「平滑」的，不能有尖點或垂直切線。例如，函數 $f(x) = |x|$ 在 $[-1, 1]$ 上連續，且 $f(-1) = f(1) = 1$，但它在 $x=0$ 處不可微分。你會發現沒有任何 $c \in (-1, 1)$ 使得 $f'(c) = 0$。

#### 2. 混淆結論中的 $c$
*   **錯誤：** 誤以為結論中的點 $c$ 是唯一的。
*   **澄清：** 羅爾定理和均值定理都只保證「至少存在一個點 $c$」。實際上，在一個給定區間內可能存在多個這樣的 $c$ 值。例如，對於 $f(x) = x^3 - x$ 在 $[-1, 1]$ 上，滿足羅爾定理條件。$f'(x) = 3x^2 - 1 = 0 \Rightarrow x^2 = \frac{1}{3} \Rightarrow x = \pm \frac{1}{\sqrt{3}}$。此時存在兩個 $c$ 值。

#### 3. 誤用區間端點的導數
*   **錯誤：** 在判斷可微分性時，將區間端點的可微分性也納入考慮。
*   **澄清：** 定理要求在**開區間** $(a, b)$ 上可微分。函數在端點 $a$ 和 $b$ 是否可微分並不影響定理的應用。這是因為定理結論中的 $c$ 點必須在開區間 $(a, b)$ 內。

#### 4. 僅依賴幾何直觀
*   **錯誤：** 僅憑視覺判斷函數圖形是否滿足條件。
*   **澄清：** 幾何解釋有助於理解定理，但在嚴格的數學推導和應用中，必須嚴格檢查函數是否滿足所有代數條件。例如，一個看起來平滑的圖形可能在某些點不可微分（如在某些理論情況下可能存在的「非標準」函數）。

### 小練習 (附詳解)

#### 小練習 1：應用羅爾定理
判斷函數 $f(x) = x^2 - 6x + 8$ 在區間 $[2, 4]$ 上是否滿足羅爾定理的條件？如果滿足，請找出所有滿足結論的 $c$ 值。

**詳解：**
1.  **檢查連續性：**
    $f(x) = x^2 - 6x + 8$ 是一個多項式函數，因此它在整個實數軸上連續，當然也在閉區間 $[2, 4]$ 上連續。
2.  **檢查可微分性：**
    $f'(x) = 2x - 6$。多項式函數的導函數也是多項式，因此 $f(x)$ 在整個實數軸上可微分，當然也在開區間 $(2, 4)$ 上可微分。
3.  **檢查端點函數值：**
    $f(2) = (2)^2 - 6(2) + 8 = 4 - 12 + 8 = 0$
    $f(4) = (4)^2 - 6(4) + 8 = 16 - 24 + 8 = 0$
    由於 $f(2) = f(4) = 0$，條件滿足。
4.  **找出 $c$ 值：**
    根據羅爾定理，存在 $c \in (2, 4)$ 使得 $f'(c) = 0$。
    $f'(c) = 2c - 6 = 0$
    $2c = 6$
    $c = 3$
    點 $c=3$ 確實在開區間 $(2, 4)$ 內。

**結論：** 函數 $f(x) = x^2 - 6x + 8$ 在區間 $[2, 4]$ 上滿足羅爾定理的所有條件，且 $c = 3$ 是滿足結論的唯一值。

#### 小練習 2：應用均值定理
對於函數 $f(x) = x^3 - 3x$ 在區間 $[0, 2]$ 上，請確認均值定理是否適用。如果適用，找出所有滿足結論的 $c$ 值。

**詳解：**
1.  **檢查連續性：**
    $f(x) = x^3 - 3x$ 是一個多項式函數，在整個實數軸上連續，因此在閉區間 $[0, 2]$ 上連續。
2.  **檢查可微分性：**
    $f'(x) = 3x^2 - 3$。多項式函數的導函數也是多項式，因此 $f(x)$ 在整個實數軸上可微分，當然也在開區間 $(0, 2)$ 上可微分。
    均值定理的所有條件都滿足。
3.  **計算割線斜率：**
    $f(0) = (0)^3 - 3(0) = 0$
    $f(2) = (2)^3 - 3(2) = 8 - 6 = 2$
    割線斜率 $m = \frac{f(2) - f(0)}{2 - 0} = \frac{2 - 0}{2 - 0} = \frac{2}{2} = 1$。
4.  **找出 $c$ 值：**
    根據均值定理，存在 $c \in (0, 2)$ 使得 $f'(c) = m$。
    $f'(c) = 3c^2 - 3$
    設置 $3c^2 - 3 = 1$
    $3c^2 = 4$
    $c^2 = \frac{4}{3}$
    $c = \pm \sqrt{\frac{4}{3}} = \pm \frac{2}{\sqrt{3}} = \pm \frac{2\sqrt{3}}{3}$
    我們需要找出落在開區間 $(0, 2)$ 內的 $c$ 值。
    $c_1 = \frac{2\sqrt{3}}{3}$。因為 $\sqrt{3} \approx 1.732$，所以 $c_1 \approx \frac{2 \times 1.732}{3} = \frac{3.464}{3} \approx 1.155$。這個值在 $(0, 2)$ 內。
    $c_2 = -\frac{2\sqrt{3}}{3}$。這個值不在 $(0, 2)$ 內。

**結論：** 函數 $f(x) = x^3 - 3x$ 在區間 $[0, 2]$ 上適用均值定理，且 $c = \frac{2\sqrt{3}}{3}$ 是滿足結論的唯一值。

### 延伸閱讀與參考

1.  **大學微積分教科書：**
    *   Stewart, James. *Calculus: Early Transcendentals*. (這是許多大學常用的標準微積分教材，對均值定理及其應用有詳細闡述。)
    *   Thomas, George B., Jr., et al. *Thomas' Calculus*. (另一本經典教材，內容豐富，例子詳盡。)

2.  **實分析 (Real Analysis) 教科書：**
    *   Rudin, Walter. *Principles of Mathematical Analysis*. (俗稱「小Rudin」，對微積分中的定理進行更嚴格、抽象的證明，均值定理在其中有基礎地位。)
    *   Apostol, Tom M. *Calculus, Vol. 1: One-Variable Calculus with an Introduction to Linear Algebra*. (提供微積分概念的嚴謹處理，適合希望深入理解數學原理的讀者。)

3.  **線上資源：**
    *   Khan Academy: 搜尋 "Mean Value Theorem" 或 "Rolle's Theorem"，提供直觀的教學影片和練習。
    *   維基百科：提供均值定理及其相關定理的詳細數學定義和證明。

這些資源能幫助讀者更深入地理解均值定理的數學基礎、證明技巧以及在更高階數學中的應用。
